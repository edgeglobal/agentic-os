# Entscheidungen-Log

Wichtige Entscheidungen mit Datum + Begruendung, damit spaeter klar ist warum etwas so gemacht wurde.

## Format pro Eintrag

```markdown
## YYYY-MM-DD — [Titel der Entscheidung]

**Worum es ging:** [Eine Zeile.]

**Optionen:** A / B / C

**Entschieden:** [Was. Eine Zeile.]

**Warum:** [Was war ausschlaggebend. 2-3 Saetze.]

**Wer beteiligt:** [Namen]
```
