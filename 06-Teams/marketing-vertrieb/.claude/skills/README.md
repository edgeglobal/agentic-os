# Team-Skills

Skills die das Team gemeinsam nutzt. Wenn ein Mitarbeiter im Workspace arbeitet und in diesem Team-Folder ist, sind diese Skills verfuegbar.

## Was sich als Team-Skill eignet

- Wiederkehrende Team-spezifische Aufgaben (z.B. Angebot-Entwurf, Liefer-Status-Mail)
- Berichts-Formate die das Team an Geschaeftsfuehrung liefert
- Onboarding-Workflows fuer neue Team-Mitglieder

## Pflege

Halbjaehrlich vom Team-Verantwortlichen pruefen: welche Skills werden genutzt, welche veralten. Veraltetes in `99-Archiv/` verschieben.

## Skill-Aufbau

Jeder Skill ist ein Sub-Folder mit `SKILL.md` drin. Schema siehe `02-Vorlagen/skill-vorlage.md` oder existierende Skills in `.claude/skills/` auf Repo-Root.
