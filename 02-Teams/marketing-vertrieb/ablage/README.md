# Team-Ablage

Fertige Outputs des Teams die **andere Teams oder die Geschaeftsfuehrung** einsehen sollen.

## Was rein gehoert

- Quartalsberichte
- Strategie-Papiere
- Praesentationen an Geschaeftsfuehrung
- Auswertungen mit firmenweiter Relevanz
- Final-Versionen von Cross-Team-Outputs

## Was nicht

- **Drafts in Bearbeitung** → in Projekt-Folder unter `projekte/<projekt>/outputs/`
- **Team-internes Wissen** → in `referenzen/`
- **Kundenspezifische Outputs** → in `03-CRM/Unternehmen/<firma>/ablage/`

## Organisation

Bei mehr als 20 Files: Sub-Folder nach Datum oder Thema:

```
ablage/
├── 2026-Q1/
├── 2026-Q2/
└── praesentationen/
    ├── 2026-05-quartal-review.pptx
    └── 2026-08-strategie-vorschlag.pdf
```

## Naming

`YYYY-MM-DD-typ-thema.md` / `.pptx` / `.pdf`

Beispiele:
- `2026-04-15-quartalsbericht-q1.md`
- `2026-06-30-h1-review.pptx`
