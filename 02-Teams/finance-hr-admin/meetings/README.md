# Team-Meetings

Interne Meetings des Teams: Standups, Wochen-Reviews, Retrospektiven, Strategie-Sessions.

> Externe Meetings (mit Unternehmen oder Personen) gehoeren in `03-CRM/Meetings/`, nicht hierhin.

## Was hier rein gehoert

- Team-Standups (taeglich/woechentlich)
- Wochen-Review oder Sprint-Planning
- Retrospektiven
- Team-Strategie-Sessions
- 1:1 zwischen Mitarbeitern im Team

## Naming

`YYYY/MM/YYYY-MM-DD-<thema>.md`

Beispiele:
- `2026/05/2026-05-12-standup.md`
- `2026/05/2026-05-15-wochen-review.md`
- `2026/05/2026-05-20-retro-q2.md`

## Mindest-Struktur

```markdown
# [Thema] — YYYY-MM-DD

**Teilnehmer:** [Namen]
**Typ:** [Standup / Review / Retro / ...]

## Was besprochen wurde

- [Bullet]

## Entscheidungen

- [Was, wer]

## Action Items

- [ ] [Was, wer, bis wann]
```

## Verknuepfung

Wenn das Meeting auf ein laufendes Team-Projekt verweist: Wikilink setzen → `[[../projekte/<projekt>]]`.
